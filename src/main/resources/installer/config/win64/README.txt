Installation - Ecolego 3.0
---------------------------------------------------------------------------

This package should contain the following files:

1. jre-6-windows-i586.exe (Sun Java Runtime Edition 1.6 Installer)
2. setup.exe (Facilia Ecolego 3.0 Installer)
3. README.txt (This file)

To install, follow these steps:

1. Install Java by clicking on jre-6-windows-i586.exe
2. Install Ecolego by clicking on setup.exe

This completes the installation of Ecolego.
